# AI-Text-Summarizer
Projekt prostej aplikacji do streszczania artykułów przy pomocy AI.
Do zbudowania projektu planuje użyć narzędzi: requests, BeautifulSoup, API modelu AI i python-dotenv.
Po wklejeniu adresu url program scrapuje z niego tekst, a potem wysyła ten tekst do AI z prośbą o wypisanie 3 najważniejszych wniosków.

A project to create a simple app for summarising articles using AI. 
I plan to use the following tools to build the project: requests, BeautifulSoup, the AI model API and python-dotenv. 
Once a URL is entered, the programme extracts the text from it and then sends this text to the AI, asking it to list the three most important takeaways.
